# Apache Atlas

Apache Atlas یک چارچوب اوپن‌سورس برای **متادیتا و حکمرانی داده** است که بیشتر با اکوسیستم Hadoop و داده‌های بزرگ (Big Data) استفاده می‌شود.

کاربردهای اصلی:

- تعریف و مدیریت اصطلاحات کسب‌وکار (Business Glossary)
- مدیریت Lineage و طبقه‌بندی داده‌ها
- ادغام با ابزارهای اکوسیستم Hadoop
