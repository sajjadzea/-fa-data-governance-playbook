# ماتریس سطوح دسترسی به داده

این ماتریس کمک می‌کند مشخص کنید چه نقشی به کدام نوع داده، چه سطح دسترسی‌ای دارد.

---

## ۱. سطح دسترسی‌ها

- `NONE`  → بدون دسترسی
- `READ`  → فقط مشاهده/دریافت داده
- `WRITE` → ایجاد/ویرایش داده
- `ADMIN` → مدیریت دسترسی دیگران + تنظیمات

---

## ۲. مثال ماتریس

| Data Domain | Data Set Example | Role: Data Steward | Role: Analyst | Role: External Researcher |
| --- | --- | --- | --- | --- |
| مشترکین آب | لیست کامل مشترکین با شناسه و آدرس | ADMIN | NONE | NONE |
| مصرف ماهانه آب (تجمیعی) | مصرف ماهانه در سطح ناحیه/محله | READ | READ | READ |
| رخدادهای شکستگی لوله | لیست رخدادها با مکان و زمان | ADMIN | READ | NONE |
| شاخص‌های بهره‌وری | KPIهای سالانه سطح استان | READ | READ | READ |

> پیشنهاد: این جدول را می‌توانید در Excel یا Google Sheets کپی کنید و برای هر واحد/سامانه جداگانه تنظیم نمایید.
