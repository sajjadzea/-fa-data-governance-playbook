# کتابچه حکمرانی داده فارسی

این پروژه یک کتابچه و جعبه‌ابزار برای حکمرانی داده در سازمان‌های ایرانی است؛ شامل Templateها، سناریوهای عملی، راهنماها و معرفی ابزارهای اوپن‌سورس.

## بخش‌های کلیدی

- [Templates](templates/data-request-form.md) برای فرم درخواست داده، موافقت‌نامه اشتراک داده و ماتریس دسترسی
- [Scenarios](scenarios/student-to-gov-data-request.md) برای نمونه مسیرهای اشتراک و درخواست داده
- [Guides](guides/quick-start-for-iran-orgs.md) برای شروع سریع و راهنمای اجرایی
- [Tools](tools/openmetadata.md) برای آشنایی با ابزارهای اوپن‌سورس حکمرانی داده

این مخزن برای نیازهای واقعی سازمان‌های ایرانی طراحی شده و می‌تواند بر اساس شرایط هر سازمان بومی‌سازی شود.
